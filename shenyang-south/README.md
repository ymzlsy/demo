# 沈阳南需求原型 Demo（4.5 数字孪生 + 4.8 预约排课）

用于 6/5 勾老师需求结论的可视化确认，**静态前端、无后端、纯需求示意**，不是正式系统。

## 运行
```bash
cd demo/prototype
node server.js          # 或 python3 -m http.server 8899
# 浏览器打开 http://127.0.0.1:8899/
```

## 覆盖范围（顶部两个模块切换）

### 4.5 数字孪生实训管控
| 页面 | 对齐的需求点 |
|---|---|
| 总览首页 | 一级先分「实训场地 / 培训场地」两入口 + 集团→校区→车间→场地层级树 |
| 段·车间数据看板 | 年度/月度切换 + KPI(参培/合格率/利用率/兑现率) + 校区参培/设备热度排行条形图 |
| 实训场地（设备级）| 占用精确到**单台设备**；**按培训任务高亮关联设备**；**三级远程控制**(单台/本室/校区关机+上电)；**自动关机规则**说明；点设备抽屉含**任务-设备隶属树**(课程→一套设备→二级→三级)+视频+历史录像 |
| 教室管控+点名 | 教员登录上课→大屏亮起；点教室进**点名流程**(工卡登录→选课程任务→参陪人员点名→开始培训→生成录像时间标签) |

### 4.8 预约排课管理
| 页面 | 对齐的需求点 |
|---|---|
| 智能排课 | **两段式步骤条**(建计划粗→添加课程→一键排课细→选老师)；设备可重复添加**打标签**；一键自动排「周几/几点」避冲突；**点课程块→选老师弹窗**(部门/电话/年度课时/总课时，正倒排、学时不足高亮、12/16/20档位) |
| 课表总览 | 日期×节次二维表，四维筛选(班级/课程/老师/教室或设备)，冲突标红，推送电子班牌 |
| 手动排课（层级筛选）| **日期→时段→形式→地点 逐层选**，选到地点层已占用项**即时标红禁选**；并发锁+两段式说明 |
| 弹性预约（月/周/个人）| 每月一练(组队选人 15min×人数) / **每周一练**(在线学习凑1学时·进度环) / 个人练功(最细15min·到点下机) |

## 交互验证点
- 智能排课：添加几门课 → 点「⚡一键智能排课」→ 看课表自动填充并避开预置冲突
- 课表总览：切换四个筛选下拉 → 表格按维度过滤
- 实训场地：点任意设备 → 右侧抽屉看详情/视频/历史
- 个人/每月预约：点绿色格子选时段（红色已占用不可选）
